# MVP Plan

## Goal

Build a WhatsApp webhook prototype that can qualify leads and support human handoff workflows.

## Milestone 1

* Webhook verification.
* Receive inbound WhatsApp messages.
* Log messages safely.

## Milestone 2

* Intent detection.
* Qualification question flow.
* FAQ responses.

## Milestone 3

* Human handoff rules.
* Lead storage.
* CRM-ready export format.

## Milestone 4

* Documentation review.
* Testing and validation.
* Demo-ready workflow.
